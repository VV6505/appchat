ChatMulti — đóng gói server.jar và client.jar
============================================

Yêu cầu: JDK 21 làm toolchain build (bytecode --source/--target 21 trong
nbproject/project.properties). Không dùng JDK 22 làm JDK mặc định khi build
nếu bạn muốn đồng bộ môi trường với JDK 21.

1) Dòng lệnh (Windows, khuyến nghị — tự chọn JDK 21 qua JAVA_HOME / JDK21_HOME):

     .\build-jars.ps1

   Hoặc sau khi đặt JAVA_HOME trỏ tới JDK 21:

     ant jars

2) NetBeans: Project Properties → Libraries → Java Platform = JDK 21.
   Clean and Build (F11), sau đó Ant target "jars":
   - Chuột phải project → Run Target → Other Targets → jars

3) Kết quả trong thư mục dist\:
   - server.jar  → java -jar server.jar   (Main: chatmulti.ui.serverui.ServerUI)
   - client.jar  → double-click hoặc java -jar client.jar (Main: chatmulti.ui.client.Client)

4) ChatMulti.jar (build mặc định của NetBeans) dùng main.class trong
   nbproject/project.properties — hiện là chatmulti.ui.client.Client.

Cấu trúc mã nguồn (UI):
   src/chatmulti/ui/serverui/ServerUI.java  — package chatmulti.ui.serverui
   src/chatmulti/ui/client/Client.java      — package chatmulti.ui.client

Cổng mặc định (sửa trong Server.java nếu cần):
  TCP 5000 — điều khiển CONNECT / JOINED_ROOM / SYSTEM
  UDP 6000 — REGISTER và ROOM_MSG (server forward)

Triển khai VPS: mở inbound TCP 5000 và UDP 6000 trên firewall / NSG Azure.
Client nhập IP public VPS, TCP 5000, UDP 6000.
