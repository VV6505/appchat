ChatMulti — đóng gói server.jar và client.jar
============================================

Yêu cầu: JDK 21 (nbproject/project.properties: javac.source / javac.target = 21).

0) NetBeans Clean and Build (F11) hoặc ant jar:
   - Luôn tạo dist\ChatMulti.jar (main.class trong project.properties).
   - Tự tạo thêm dist\server.jar (Main ServerUI) ngay sau đó — không ghi đè ChatMulti.jar.

1) Chỉ gói server (không bắt buộc chạy jar/ChatMulti trước), ví dụ máy chỉ compile rồi đóng server:

     ant server-jar

   File: dist\server.jar — chạy trên máy chủ: java -jar server.jar
   (Main: chatmulti.ui.ServerUI — giao diện Swing; máy Azure Windows có desktop
   hoặc RDP thì chạy bình thường; Linux không GUI cần cấu hình hiển thị ảo nếu dùng.)

2) Gói cả server + client:

     ant jars

   Hoặc NetBeans: Run Target → Other Targets → server-jar hoặc jars.

3) Kết quả trong thư mục dist\:
   - server.jar  → java -jar server.jar   (Main: chatmulti.ui.ServerUI)
   - client.jar  → double-click hoặc java -jar client.jar (Main: chatmulti.ui.Client)

4) ChatMulti.jar (build mặc định) dùng main.class trong
   nbproject/project.properties — hiện là chatmulti.ui.Client.
   Cùng thư mục dist\ luôn có thêm server.jar sau mỗi lần build JAR chuẩn (xem mục 0).

Cấu trúc mã nguồn (UI):
   src/chatmulti/ui/ServerUI.java  — package chatmulti.ui
   src/chatmulti/ui/Client.java    — package chatmulti.ui

Cổng mặc định (sửa trong Server.java nếu cần):
  TCP 5000 — điều khiển CONNECT / JOINED_ROOM / SYSTEM
  UDP 6000 — REGISTER và ROOM_MSG (server forward)

Triển khai VPS: mở inbound TCP 5000 và UDP 6000 trên firewall / NSG Azure.
Client nhập IP public VPS, TCP 5000, UDP 6000.
